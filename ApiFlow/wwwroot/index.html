Insert generator into a value

Click a value field, then a generator to insert its token. Resolved fresh on each Run.

Move a node by dragging its title bar.

Wire by dragging from a node's right ● output pin to another node's left ● input pin.

Remove a wire by clicking it, or clicking the connected input ● pin.

The body is key/value fields, serialized by Content-Type: application/json → JSON object, x-www-form-urlencoded → form string (other types use a raw text box).

Outputs can wire into header and body-field values (each has a left ● pin), and into Transform nodes. One output can feed several targets.

If an output is an object, set its response path (or pick a key from the ▾ dropdown after a run) so only that value flows on.

A wired value appends to what you typed (so Bearer keeps its prefix), or place it precisely with the {{name}} chip.

Use + obj to nest an object field (e.g. Amount → { Value, Currency }). Values are sent as strings unless they are true/false/null or JSON {…}/[…].

Run all executes nodes in dependency order, piping wired values along.
